# 搜索Github仓库 常用搜索关键字总结

  1. in:name xxx 在名字中搜索xxx，如in:name vue，表示名字中含有 vue的仓库
  2. in:description xxx 在描述中搜索xxx，如in:description vue，表示描述中含有 vue的仓库
  3. in:readme xxx 在 readme 中搜索xxx，如in:readme vue全家桶，表示描述中含有 vue全家桶的仓库
  4. srars:>xxx 按 stars的个数搜索，数字越大，表示越受欢迎。如stars:>1000，表示仓库的 stars 数要大于 1000 个。
  5. fork:>xxx 按 fork 的个数搜索，数字越大，表示拷贝过的人越多，有热度。如fork:>1000，表示仓库的 fork 数要大于 1000 个
  6. pushed:>YYYY-MM-DD 按最新更新时间搜索，更新时间越靠近当前，表示仓库仍旧活跃。日期如pushed:>2020-09-01，表示仓库的最新更新时间在2020-09-01后。
  7. user:xxx 按作者名搜索，如user:tnfe，表示搜索 tnfe 的所有仓库
  8. language:xxx 按语言搜索，如language:javascript，表示搜索的仓库的语言使用 javascript
  9. size:>xxx 按仓库的大小搜索，其中的单位是 k，如size:>5000，表示仓库大小要大于5000K，即 5M。

以上的搜索方式可以组合叠加使用，注意各搜索关键字和搜索条件不要有空格。

更多的github 搜索可查看在 github上搜索仓库 (opens new window)
