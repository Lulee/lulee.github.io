# 父传子

  1. props

    通过子组件的 props 定义属性，然后通过属性将父组件数据传递子组件

       <div id="app">
        <h1>我是父组件</h1>
        <!-- 注：props 在 html中驼峰式的命名要变为短横线式,因为 html 标签的属性大小写不明感 -->
        <son-comp :father-data="data"></son-comp>

    </div>
    <script src="./lib/vue.js"></script>
    <script>
        Vue.component('son-comp', {
            props: ['fatherData'],
            template: `<div>  
                        <h1>我是子组件</h1>
                        <p>子组件要接收的数据:---{{fatherData}}</p>
                       </div>`
        })
        var app = new Vue({
            el: "#app",
            data: {
                data: "来自父组件的数据"
            },
        });
    </script>
